# Concept

Detect Motion, Distance and stream live Raspberry Pi camera using websockets.

# Prerequisites

1. Raspberry Pi + Camera
2. PIR Motion Sensors
3. ToF (Time-of-Flight) VL53L0X Distance Sensors
